## p4xt3r - goutils

Вспомогательные утилиты при работе с разными проектами

#### Установка

На данные момент актуальная версия: v0.12.4

```shell

go get github.com/algrvvv/goutils@v0.12.4

```

Утилиты включают в себя: 
- работу со строками
- работу с датами
- работу с json файлами
- работу с массивами
- и другое
